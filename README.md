# Boztaş İnşaat Yapı - Tanıtım Sitesi

Modern, responsive ve tek sayfalık inşaat şirketi tanıtım sitesi.

## 🚀 Özellikler

- ✅ Modern ve profesyonel tasarım
- ✅ Tam responsive (mobil, tablet, masaüstü)
- ✅ Smooth scroll animasyonları
- ✅ Galeri lightbox özelliği
- ✅ Hamburger menü (mobil)
- ✅ SEO uyumlu
- ✅ Vercel'de deploy edilmeye hazır

## 📁 Proje Yapısı

```
boztasinsaatyapi/
├── index.html          # Ana HTML dosyası
├── script.js           # JavaScript fonksiyonları
├── img/                # Galeri fotoğrafları klasörü
└── README.md           # Bu dosya
```

## 🛠️ Teknolojiler

- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript
- Google Fonts (Montserrat)

## 🎨 Tasarım

- **Ana Renk**: #F2C14E (Altın Sarısı)
- **Arka Plan**: #1E1E1E (Koyu Gri)
- **Yazı Rengi**: Beyaz
- **Font**: Montserrat

## 📱 Bölümler

1. **Header** - Logo ve navigasyon menüsü
2. **Hero** - Ana başlık ve çağrı butonları
3. **Hizmetler** - 8 farklı hizmet kartı
4. **Hakkımızda** - Şirket bilgileri ve "Neden Biz?" bölümü
5. **Galeri** - Proje fotoğrafları (filtrelenebilir)
6. **İletişim** - Telefon, WhatsApp ve Google Maps
7. **Footer** - İletişim bilgileri ve telif hakkı

## 🚀 Deployment

### Vercel'e Deploy Etme

1. Projeyi GitHub'a yükleyin
2. [Vercel](https://vercel.com) hesabınıza giriş yapın
3. "New Project" butonuna tıklayın
4. GitHub repository'nizi seçin
5. Deploy butonuna tıklayın

Vercel otomatik olarak statik siteyi algılayacak ve deploy edecektir.

### Manuel Deploy

```bash
# Projeyi klonlayın
git clone <repository-url>

# Vercel CLI ile deploy edin
vercel
```

## 📝 Notlar

- Telefon numaraları placeholder olarak bırakılmıştır. Gerçek numaraları `index.html` dosyasında güncelleyin.
- WhatsApp linkini gerçek numara ile güncelleyin: `https://wa.me/90XXXXXXXXXX`
- Google Maps embed URL'ini gerçek konum ile güncelleyin.
- Galeri fotoğraflarını `img/` klasörüne ekleyip `index.html` dosyasındaki görsel URL'lerini güncelleyin.
