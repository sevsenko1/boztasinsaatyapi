# Galeri Fotoğrafları

Bu klasöre proje fotoğraflarınızı ekleyebilirsiniz.

## Önerilen Dosya İsimleri:
- `insaat-1.jpg`, `insaat-2.jpg` vb. (İnşaat projeleri için)
- `cati-celik-1.jpg`, `cati-ahsap-1.jpg` vb. (Çatı işleri için)
- `tadilat-1.jpg`, `tamirat-1.jpg` vb. (Tadilat & tamirat için)

## Not:
Şu anda site Unsplash placeholder görselleri kullanıyor. Gerçek fotoğraflarınızı ekledikten sonra `index.html` dosyasındaki görsel URL'lerini güncelleyin.

Örnek:
```html
<img src="img/insaat-1.jpg" alt="İnşaat Projesi" class="...">
```
